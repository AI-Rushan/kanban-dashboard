📅 Google Calendar

Синхронизируй задачи с Google Calendar. Все события из выбранного календаря появятся в Бэклоге. Задачи с дедлайном создадут события в календаре.

Google Cloud Console → APIs & Services → Credentials → OAuth 2.0 Client IDs.
В «Authorized JavaScript origins» добавь http://localhost (или адрес, откуда открываешь файл).
⚠ Файл нужно открывать через http://localhost, не через file://.
✓ Авторизация успешна
Выбери, с каким календарём синхронизировать задачи
✓ Подключено
Календарь:
При синхронизации:
• События из календаря (32 дня) → Бэклог
• Задачи с дедлайном → события в календаре
• Повторяющиеся события разворачиваются
Перезапишет ВСЕ задачи с дедлайном в Calendar без проверки дат. Используй если обычная синхронизация не помогает.
