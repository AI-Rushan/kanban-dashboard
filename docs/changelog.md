# История изменений — «Канбан-дашборд»

---

## v1.0 — 2026-06-11

Первый полноценный релиз с полным набором функций.

### Реализовано

**Канбан-доска**
- Четыре колонки: Backlog → Todo → Doing → Done
- Перемещение карточек: перетаскиванием (drag & drop) и кнопками ← →
- Счётчики задач в заголовках колонок
- Статистика в шапке: количество задач по статусам + просроченные

**Задачи**
- Поля: текст, направление, дедлайн, время дедлайна, описание
- Автоматическая фиксация `completedAt` при переводе в «Сделано»
- Визуальная индикация дедлайна: просрочено / сегодня / скоро / в норме
- Ссылки в описании автоматически кликабельны (`linkify`)

**Направления (Areas)**
- 8 предустановленных направлений: Поиск работы, Мама/Врачи, Дом, Обучение ИИ, Пет-проект, Бизнес, Дача, Хобби
- Динамическое управление: добавить, редактировать, удалить
- Палитра из 12 цветов
- Фильтрация по направлению через чипы в тулбаре

**Сортировка**
- Независимая для каждой колонки
- Варианты: по дедлайну (↑↓), по направлению, по дате завершения

**Архив (`archive.html`)**
- Автоматический переход задач `done → archive` после текущей ISO-недели
- Группировка по направлениям со сворачиванием
- Фильтрация и сортировка
- Статистика: вовремя / с опозданием / без дедлайна
- Бейджи на каждой задаче

**Уведомления**
- Браузерные push-уведомления о просроченных и срочных задачах
- Защита от спама: проверка раз в час

**Экспорт / Импорт**
- Экспорт в JSON-файл (`kanban_backup_ДАТА.json`)
- Импорт с заменой данных без перезагрузки страницы
- Автоматическая миграция полей при импорте
- Надёжная реализация через статический `<label> + <input type="file">` (работает во всех браузерах)

**Google Calendar — двусторонняя синхронизация**
- OAuth 2.0 через Google Identity Services (popup flow)
- Выбор конкретного календаря из списка
- Автосинхронизация при открытии страницы
- Ручная синхронизация кнопкой 🔄
- Статус задачи → цвет события (серый / синий / оранжевый / зелёный)
- Связь через `extendedProperties.private.taskId`
- Разрешение конфликтов: last-write-wins по `lastModified`
- Окно синхронизации: текущая неделя + 32 дня
- Принятые приглашения из Calendar → задачи в Backlog
- Таймаут авторизации 90 секунд (защита от вечного зависания)

**Запуск**
- `launch.command` — двойной клик запускает локальный сервер и открывает браузер
- Автоматическое освобождение порта 8080 если занят
- Корректный останов сервера при закрытии окна (`trap EXIT`)

## Известные ограничения v1.0

- Только macOS (`launch.command` — bash-скрипт)
- OAuth в тестовом режиме Google (нужно добавлять пользователей вручную)
- Токен Google живёт ~1 час (после истечения нужен новый попап)
- Safari требует ручного разрешения попапов для localhost
