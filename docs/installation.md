# Установка — «Канбан-дашборд»

Версия: 1.0 | Дата: 2026-06-11

---

## Требования

- macOS (любая версия с 2020 года)
- Python 3 — предустановлен на macOS 12 и новее; проверить: открой Терминал и введи `python3 --version`
- Браузер: Safari или Chrome

---

## Вариант А — базовая установка (без Google Календаря)

Это самый простой вариант. Подходит для дочерей и для любого нового компьютера.

**Шаг 1.** Создай папку `kanban` в удобном месте (например, на Рабочем столе):
```
~/Desktop/kanban/
```

**Шаг 2.** Скопируй в папку два файла:
- `dashboard.html`
- `archive.html`

**Шаг 3.** Дважды кликни на `dashboard.html` — дашборд откроется в браузере.

Всё. Больше ничего не нужно.

---

## Вариант Б — полная установка (с Google Календарём)

Google Calendar требует, чтобы страница открывалась через `http://localhost`, а не через `file://`. Для этого нужен скрипт запуска.

**Шаг 1.** Создай папку `kanban` на Рабочем столе:
```
~/Desktop/kanban/
```

**Шаг 2.** Скопируй в папку три файла:
- `dashboard.html`
- `archive.html`
- `launch.command`

**Шаг 3.** Сделай скрипт исполняемым. Открой Терминал и выполни:
```bash
chmod +x ~/Desktop/kanban/launch.command
```
(Это нужно сделать только один раз при первой установке.)

**Шаг 4.** Дважды кликни на `launch.command`. Откроется чёрное окно терминала и браузер с дашбордом.

## Настройка Google Calendar (опционально)

Нужна только если хочешь синхронизировать задачи с Google Календарём. Без этого дашборд работает полностью.

### Шаг 1 — создать проект в Google Cloud Console

1. Открой [console.cloud.google.com](https://console.cloud.google.com)
2. Нажми на меню проектов вверху → **«Новый проект»**
3. Назови проект `kanban` → **«Создать»**

### Шаг 2 — включить Google Calendar API

1. В левом меню: **APIs & Services → Library**
2. Найди **Google Calendar API** → нажми **Enable**

### Шаг 3 — создать OAuth Client ID

1. В левом меню: **APIs & Services → Credentials**
2. Нажми **«+ Create Credentials» → «OAuth client ID»**
3. Тип приложения: **Web application**
4. Название: `kanban`
5. В поле **Authorized JavaScript origins** добавь: `http://localhost:8080`
6. Нажми **«Create»**
7. Скопируй **Client ID** (длинная строка вида `...apps.googleusercontent.com`)

### Шаг 4 — настроить экран согласия и добавить себя как тестового пользователя

1. **APIs & Services → OAuth consent screen**
2. User Type: **External** → **Create**
3. Заполни: App name = `kanban`, User support email = твой Gmail
4. Нажми **Save and Continue** (на каждом шаге)
5. В разделе **Test users**: нажми **+ Add users** → добавь свой Gmail-адрес
6. Сохрани

### Шаг 5 — подключить в дашборде

1. Открой дашборд через `launch.command`
2. Нажми кнопку **📅** в тулбаре
3. Вставь Client ID → нажми **«Подключить»**
4. Разреши всплывающее окно (если браузер спрашивает)
5. Выбери нужный календарь из списка → **«Сохранить»**

## Разрешить Safari открывать всплывающие окна (для Google Auth)

Safari по умолчанию блокирует попапы. Чтобы OAuth работал:

1. В Safari: **Настройки → Сайты → Всплывающие окна**
2. Найди `localhost` в списке → поставь **«Разрешить»**

После установки убедись, что всё работает:

- [ ] Дашборд открывается в браузере
- [ ] Можно создать задачу
- [ ] Задача сохраняется после закрытия и повторного открытия браузера
- [ ] (Опционально) Синхронизация с Google Calendar работает — кнопка 🔄 появилась в тулбаре
