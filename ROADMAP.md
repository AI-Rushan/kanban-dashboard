# Роадмап «Канбан-дашборд»

Планы развития продукта. Приоритет — личное использование, без серверной части и групповой работы.

---

## v1.2 — Быстрые улучшения

### 🪟 Windows-совместимость
- Скрипт `launch.bat` или PowerShell для запуска на Windows
- Поведение идентично `launch.command` на macOS: запуск сервера → открытие браузера
- Код `dashboard.html` не меняется

### 📎 Вложения файлов
- Папка `kanban/files/` рядом с `dashboard.html`
- Файлы из этой папки доступны через `http://localhost:8080/files/имя_файла`
- В карточке: поле для ссылок на файлы, кликабельный список вложений
- Файлы добавляются вручную в папку, ссылка копируется в карточку

### 🌙 Тёмная тема
- Переключатель в тулбаре (☀️ / 🌙)
- Автоматически следует системной настройке macOS/Windows
- Сохраняется в localStorage

### 🔍 Полнотекстовый поиск
- Поле поиска в тулбаре
- Поиск по названию и описанию задачи в реальном времени
- Фильтрует карточки на всех колонках одновременно

---

## v1.3 — Продуктивность

### ⌨️ Горячие клавиши
- `N` — создать новую задачу
- `Esc` — закрыть модальное окно
- `←` / `→` — переместить карточку между колонками
- `?` — показать список всех шорткатов

### ✅ Подзадачи / чеклист
- Список чекбоксов внутри карточки
- Прогресс-бар на карточке (`3/5 выполнено`)
- Возможность конвертировать подзадачу в отдельную карточку

### 📊 Статистика и аналитика
- Задачи выполнено за неделю / месяц
- Процент выполнения в срок vs с опозданием
- Разбивка по направлениям
- Доступна на странице архива

### 📝 Markdown в описании
- Форматирование в поле описания: **жирный**, *курсив*, списки, `код`, ссылки
- Рендерится при просмотре карточки, редактируется как текст

---

## v2.0 — Большие фичи

### 🔁 Повторяющиеся задачи
- Настройка: ежедневно / еженедельно / ежемесячно
- Задача автоматически пересоздаётся в Backlog после выполнения
- Синхронизация с Google Calendar (recurring events)

### 🗂️ Несколько досок
- Несколько независимых канбан-досок (Работа, Личное, Дача...)
- Переключение через выпадающий список в тулбаре
- Каждая доска — отдельный набор задач и направлений в localStorage

### 📅 Встроенный вид «Календарь»
- Месячный / недельный вид задач прямо в дашборде
- Не требует подключения Google Calendar
- Drag & drop для переноса дедлайнов

### 📤 Экспорт в CSV и PDF
- Экспорт списка задач в CSV для анализа в Excel/Numbers
- Экспорт в PDF для печати или отчётов
- Выбор фильтра перед экспортом (статус, направление, период)

## v3.0 — Амбиции

### ☁️ Облачный бэкап
- Автоматическое сохранение данных в Google Drive или iCloud
- Восстановление данных на новом компьютере без ручного экспорта/импорта
- Бэкап по расписанию (ежедневно)

### 📱 Мобильный вид
- Адаптивный дизайн для смартфона и планшета
- Доступ через локальную сеть Wi-Fi (с телефона на тот же localhost)
- Swipe для перемещения карточек

### ⏱️ Трекер времени
- Кнопка «Старт / Стоп» таймера на карточке
- Лог потраченного времени с историей
- Сводка в статистике: сколько времени уходит на каждое направление

### 🤖 AI-помощник
- Разбить большую задачу на подзадачи одной кнопкой
- Предложить реалистичный дедлайн на основе объёма
- Сгенерировать описание задачи по короткому заголовку
- Работает через Anthropic API (Claude)

---

## Принципы развития

- Один HTML-файл — никаких фреймворков и сборщиков
- Данные хранятся локально, не передаются третьим лицам (кроме Google Calendar по явному запросу)
- Установка — двойной клик, никакого терминала для базового использования
- Совместимость: macOS и Windows
- Личное использование — никакой групповой работы и авторизации
